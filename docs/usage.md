# 使用指南

## 启动机器人

在配置好 `settings.toml` 和 `acl.toml` 后，再次运行程序。

如果配置正确，您应该会看到类似以下的日志：

```
INFO Starting TeamSpeakClaw v0.1.0
INFO Bot ready. Listening for events.
```

此时，机器人应该已经连接到您的 TeamSpeak 服务器。

## 交互方式

您可以通过以下方式与机器人交互：

1.  **频道聊天**: 在频道发送消息，带上触发前缀。
    -   默认前缀: `!tsclaw`, `!bot`, `@TSClaw`
    -   例如: `!bot 播放周杰伦的夜曲`

2.  **私聊 (推荐)**: 双击机器人进行私聊。
    -   私聊通常不需要前缀（取决于 `respond_to_private` 设置）。
    -   例如: `踢掉那个叫 User123 的人`

## 可用技能 (Skills)

机器人目前支持以下技能（取决于您的权限配置）：

### 🎵 音乐控制
- **点歌**: "播放 [歌名]"
- **切歌**: "下一首", "切歌"
- **暂停/继续**: "暂停音乐", "继续播放"

*(注意: 需要配合 TS3AudioBot 使用)*

### 🛡️ 管理功能
- **踢出用户**: "把 UserA 踢出频道"
- **封禁用户**: "封禁 UserB"
- **移动用户**: "把 UserC 移动到 [频道名]"

### ℹ️ 信息查询
- **查询在线用户**: "现在谁在线？"
- **查询服务器状态**: "服务器运行多久了？"

## 常见问题

- **机器人没反应？**
    -   检查是否使用了正确的前缀。
    -   检查后台日志是否有报错。
    -   确认 LLM API Key 是否正确且有余额。

- **提示“没有权限”？**
    -   检查 `acl.toml` 中的配置，确认您的用户组 ID 是否在允许的规则中。
