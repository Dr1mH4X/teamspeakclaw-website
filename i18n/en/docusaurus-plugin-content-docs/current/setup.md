# Download & Installation

## 1. Download

Please visit the GitHub Releases page to download the latest version of TeamSpeakClaw:

[**Download Latest Version**](https://github.com/Dr1mH4X/TeamSpeakClaw/releases/latest)

Select the appropriate file for your operating system (Windows, Linux, macOS).

## 2. Installation

TeamSpeakClaw is a standalone binary application that requires no complex installation process.

1. Extract the downloaded archive into a folder.
2. Ensure you have read/write permissions for that folder.

## 3. First Run

When you run the program for the first time, it will automatically generate default configuration files in the `config/` directory.

- **Windows**: Double-click `teamspeakclaw.exe`
- **Linux/macOS**: Run `./teamspeakclaw` in the terminal (you may need to add execution permission first: `chmod +x teamspeakclaw`)

The program will prompt that the configuration file has been generated, and then may exit with an error due to missing configuration (such as TS server password or API Key). This is normal. Please refer to the [Configuration Guide](/docs/configuration) for configuration next.
